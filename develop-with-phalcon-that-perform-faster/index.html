超高速PHPフレームワークPhalconを使ったWeb制作

Falconと間違えやすいですよね、Phalcon。
インストール時にこの勘違いは致命的です。。

http://phalconphp.com/ja/

爆速とよばれるPHPフレームワークPhalcon

最近Phalconを使ってWeb制作をしていますが非常に高速に動作し、
仕組みもいわゆるフレームワークの概念から大きく外れておらず
開発のやりがいのある印象です。

Phalconがどれだけ早いかというのは、以下のサイトにベンチマークが掲載されています。

[https://docs.phalconphp.com/ja/latest/reference/benchmark/hello-world.html:title]

（2015/08/04 リンク更新）

ちゃんと、前提条件を明記してくれていたベンチで信頼性が高めですね。
手元でも似たようなベンチ結果が出ました。

Phalconを使うようになった理由はいくつかあります。

  • Core技術がC言語で書かれているが故、（他のPHPフレームワークと比較して）高速に動作する。
  • 今後の広がりがある程度見えること（廃れていくニッチなフレームワークにはならなさそう）

困っていることといえば、

  • 日本語翻訳チームに参加を申し込んだのですが、もうメンバーいっぱいなんだよと断られたにもかかわらず
    日本語ドキュメントがなかなか整理されない現状 ・・・orz

日本にももっと広めたいのに。。

（そして日本語ドキュメントは整理されていない現状orz）

ということもあり、自分のブログで紹介することで少しでもPhalconが認知され、
少しでも広く使われるようになることを期待して記事を書くことにしました。

では、早速インストール！

インストール

OS: Linux (CentOS release 6.5 (Final)) ※vagrantを使用したMac上のローカル環境

PHPのインストール

PhalconはPHP 5.3.6以上推奨の模様。
CentOSのbaseはこれよりも低い可能性があるため、remiからダウンロードします。

必要なパッケージ類のインストールのために
remi, epel, rpm-forgeレポジトリを追加します。

サードパーティ製のリポジトリを追加するので、優先度をつけるyum-propertiesを導入する。
(詳細はこちらを参照させていただきました)

yum install yum-plugin-priorities

# epel
$ rpm -Uvh http://ftp-srv2.kddilabs.jp/Linux/distributions/fedora/epel/6/x86_64/epel-release-6-8.noarch.rpm

# remi
$ rpm -Uvh http://rpms.famillecollet.com/enterprise/remi-release-6.rpm

# rpmforge
$ rpm -Uvh http://pkgs.repoforge.org/rpmforge-release/rpmforge-release-0.5.3-1.el6.rf.x86_64.rpm

必要なパッケージ類のインストール

yum install enablerepo=remi php-devel php-mysql gcc libtool git

php -vしてバージョンを確認。

phalconのダウンロード&インストール

cd /usr/local/src/
git clone git://github.com/phalcon/cphalcon.git
cd cphalcon/build
./install #←別ディレクトリからの実行はエラーがでます

PHPの設定

PHPモジュールにPhalconの動的ライブラリを読み込ませます。

$ vi /etc/php.d/phalcon.ini
extension = phalcon.so

HTTPサーバー再起動
service httpd restart

Phalcon Developer Toolsのインストール

Composerからインストールする

cd /usr/local/shareなどで

curl -s http://getcomposer.org/installer | php

同階層でcomposer.jsonを作成

{
    "require": {
        "phalcon/devtools": "dev-master"
    }
}

composer installを実行

php composer.phar install

cd /vender/phalcon/devtools以下のphalcon.shを実行

ln -s ~/devtools/phalcon.php /usr/bin/phalcon
chmod ugo+x /usr/bin/phalcon

phalconとコマンドを打って、

# phalcon

Phalcon DevTools (1.2.4)

Available commands:
  commands (alias of: list, enumerate)
  controller (alias of: create-controller)
  model (alias of: create-model)
  all-models (alias of: create-all-models)
  project (alias of: create-project)
  scaffold
  migration
  webtools

と表示されればPhalconのインストールは完了！
